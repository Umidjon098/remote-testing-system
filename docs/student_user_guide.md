# Student User Guide (MVP)

## Login

- Open `/login` and sign in (or sign up).

## Take a test

- Open `/student/tests`.
- Pick a test and click **Open**.
- Click **Start test** to begin.
- A timer is shown; when time is up, the attempt auto-submits.

## View results

- Open `/student/results`.
- You can continue an in-progress attempt from this page.
